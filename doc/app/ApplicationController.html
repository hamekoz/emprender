    class ApplicationController

    ApplicationController es el controlador principal de Emprender Provee metodos generales a todos los controladores y renderiza la pagina inicial y la de informacion acerca de Emprender

    Public Instance Methods

    acerca() click to toggle source

    Renderiza la pagina Acerca, recolecta informacion de toda Institucion, Informacion y Vinculo cuyo atributo visible sea verdadero

    # File app/controllers/application_controller.rb, line 40
def acerca
  @informaciones = Informacion.visibles
  @instituciones = Institucion.visibles
  @vinculos = Vinculo.visibles

  add_crumb "Inicio", root_path
  add_crumb "Acerca de ..."

  respond_to do |format|
    format.html # index.html.erb
  end
end
